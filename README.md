# Movilidad Urbana y Productividad Económica en LATAM

Este proyecto analiza la relación entre los niveles de congestión vehicular, tiempos de viaje y retrasos con indicadores de productividad económica, como el PIB per cápita y las tasas de desempleo, en las principales ciudades de Latinoamérica.

## 📌 Objetivo del Proyecto
Identificar patrones y correlaciones que permitan entender cómo la eficiencia del transporte urbano impacta directamente en el desarrollo económico regional. 

## 📊 Datos y Metodología
El análisis se centra en:
- **Movilidad:** Índices de congestión y tiempos de traslado promedio.
- **Economía:** PIB per cápita, niveles de empleo y productividad por hora.
- **Alcance:** Principales metrópolis de América Latina (CDMX, Bogotá, São Paulo, Santiago, Lima, etc.).

## 🛠️ Tecnologías Utilizadas
- **Python:** (Pandas, Matplotlib, Seaborn) para la limpieza y visualización de datos.
- **Jupyter Notebooks:** Para el desarrollo del flujo de análisis.
- **SQL:** Para la consulta y estructuración de bases de datos.
- **Looker Studio / Tableau:** (Si aplica) Para la creación de dashboards interactivos.

## 📂 Estructura del Repositorio
- `/data`: Conjuntos de datos utilizados (o enlaces a las fuentes originales).
- `/notebooks`: Scripts de limpieza y análisis exploratorio (EDA).
- `/visualizations`: Gráficos y reportes generados.
